# Bastille

Currently under development.
Inspired by the classic Minecraft "prisons" game mode.

Features include:
- Personal Mine Generation
- Modular Mine Upgrades 
- Enchants System
- Quests System
- Skills System
- Abilities System
- Minimap System
- Potions System
- Backpack System
- Crates System
- Overhauled Crafting
- Custom NPCs
- Region Management
